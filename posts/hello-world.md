# 你好世界 — 我的博客开张了

折腾了一晚上，总算是把博客搭起来了。

## 为什么是现在

做前端这些年，踩过的坑、趟过的雷其实不少。工单系统的离线重连、Excel 百万级导出优化、前车查找的算法设计…… 每一个都是花了不少时间才搞定的。但这些东西一直留在脑子里，时间久了就忘了细节。

写下来，既是为了自己以后能翻，也是为了给遇到同样问题的同行一个参考。

## 为什么是 VitePress

选框架的时候其实纠结了一下：

| 框架 | 优点 | 缺点 |
|------|------|------|
| VitePress | 极快、Vue 原生、零配置 | 不算是专门的博客框架 |
| Nuxt Content | 功能强、博客特性好 | 太重，有点杀鸡用牛刀 |
| Astro | 性能好、生态丰富 | 得学新语法 |

最后还是选了 **VitePress**。原因很简单——我本来就是 Vue 开发者，VitePress 的体验和 Vue SFC 开发完全一致，想加什么 Vue 组件就直接写。而且它基于 Vite，开发体验是真的快。

## 关于这个主题

暗色科幻风是自己写的。核心元素：

- 深空蓝黑底色 + 微妙的网格纹理
- 青色到紫色的渐变强调色
- 玻璃拟态（backdrop-filter）卡片
- 首页的光晕动效和打字机效果
- 氢气般的发光文字

所有样式都是 CSS，没额外引入任何 UI 库。

## 接下来会写什么

- **工单系统实战** — 离线重连、消息推送、Excel 导出的性能优化
- **Vue 3 深入** — Composition API 设计模式、组件封装思路
- **工程化** — Git 工作流、CI/CD 搭建、项目架构演进
- **随手记** — 遇到的坑和解决方案
